# ⏰ Deadline Rescue AI

> **Paste any message. Know exactly what to do.**
> Built for Vibe2Ship Hackathon · Problem Statement 1 · Powered by Google Technologies

---

## 🎯 The Problem

Indians miss deadlines every day — electricity bills get disconnected, assignments get zero marks, job applications expire — because deadline messages are buried in WhatsApp, SMS, and email. There is no single tool that reads any message and tells you exactly what to do, right now.

---

## 💡 The Solution

Paste any SMS, WhatsApp message, email, or screenshot. Deadline Rescue AI's autonomous Gemini agent extracts the deadline, scores urgency, builds a step-by-step action plan, calculates when you need to leave, and saves everything to your Firebase dashboard — all in one agent loop, zero manual steps.

---

## 🤖 Agentic Architecture

This is not a chatbot. It is a genuine autonomous multi-turn agent built on Gemini 2.5 Flash native function calling.

**How the agent loop works:**
Zero visible errors to the user. Fallback is completely transparent.

---

## 🔧 Google Technologies Used

| Technology | Usage |
|---|---|
| **Gemini 2.5 Flash** | Primary agent — native function calling, multi-turn tool loop, image OCR of screenshots |
| **Firebase Firestore** | Persistent Mission Control dashboard — deadlines saved across sessions |
| **Google Maps** | Directions link from leave-by calculator |

## ✨ Features

| Feature | Description |
|---|---|
| 📋 Multi-format input | Paste text, upload screenshot, or speak your deadline |

## 🚀 Setup

```bash
git clone https://github.com/Prem36Kumar/Deadline-Rescue-AI
cd deadline-rescue
npm install
cp .env.local.example .env.local
```

Add to `.env.local`:

```env
GEMINI_API_KEY=your_gemini_key
GROQ_API_KEY=your_groq_key
ORS_API_KEY=your_ors_key
NEXT_PUBLIC_FIREBASE_API_KEY=your_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
```

```bash
npm run dev
```
